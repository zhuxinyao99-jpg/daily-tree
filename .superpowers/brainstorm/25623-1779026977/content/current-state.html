梳理 Landing Page 和应用的主要问题

🎯 Landing Page 核心问题

当前状态

  • ✗ "你的森林在呼吸" 下面的树还是圆形
  • ✗ 需要真实树形（有树干、树杈）
  • ✗ "一年的一瞬间" 树冠树干分离
  • ✗ 三个阶段在手机上显示不完整
  • ✗ 整体偏沉闷，缺少四季效果展示
  • ✗ 英文字体不够优美（缺艺术字）

优化目标

  • ✓ 树的视觉真实化（写实树形）
  • ✓ 展示四季变化效果
  • ✓ 提升色彩和布局的质感
  • ✓ 手机显示完整优化
  • ✓ 英文字体提升审美
  • ✓ 用户一眼看出 app 有意义

核心思想： Landing page 这一页是宣传页面，需要兼具美感和实用性。让用户感受到这是一个"活着的树"，有季节变化，能看到坚持到后面的效果。色彩和画面布局要有质感，点进 app 就有安静的感觉。

📱 应用内树的改进

当前

  • ✗ 打开即显示完整的树
  • ✗ 无打卡/选时间年份功能
  • ✗ 卡片不能交互
  • ✗ 树无法旋转
  • ✗ 背景固定

优化

  • ✓ 树从第一条记录开始逐渐长大
  • ✓ 完整的打卡/时间选择功能
  • ✓ 卡片交互（可点击、展开等）
  • ✓ 树可手动旋转（3D 建模）
  • ✓ 背景根据天气变化

🌳 树的形象是最重要的部分

整个 app 的核心就是"树"。树的视觉表现需要：

  • 真实感（有树干、树杈、根系）
  • 生长感（从小到大的渐进变化）
  • 季节性（四季变化明显）
  • 交互性（可旋转、可查看细节）
  • 品质感（高端的视觉呈现）
